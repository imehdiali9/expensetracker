# 🚀 Smart Expense Tracker

A modern, responsive expense tracking dashboard designed to help users manage their finances with clarity and control.

Built with a production-style frontend stack and deployed on the web.

---

## 🌐 Live Demo
👉https://expensetracker-omega-seven.vercel.app/
---

## ✨ Features

✅ Add Income & Expenses  
✅ Create Custom Categories  
✅ Set Budgets  
✅ Visual Analytics (Charts)  
✅ Transaction History  
✅ Dark Mode 🌙  
✅ Fully Responsive UI  
✅ Clean Dashboard Layout  

---

## 🧠 Tech Stack

**Frontend**
- React (Vite)
- Tailwind CSS
- Recharts

**Version Control**
- Git + GitHub

- Vercel
